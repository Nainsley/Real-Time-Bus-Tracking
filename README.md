# Real-Time-Bus-Tracking

Description: Using a map to pinpoint the bus stops for the routes near Harvard and MIT. The map markers are added to the map along the way.

How to Run: In order to run view the html files.

Roadmap for improvement: I wish to add more data around the bus routes to update more frequently.
